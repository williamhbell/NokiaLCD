
(1) Use raspi-config to enable the SPI module.

(3) Install the dependencies
sudo apt-get -y install python-dev python-netifaces

(3) Shutdown pi
sudo shutdown -h now

(4) Connect jumper wires

 7 RST        11              GPIO17
 8 LED        12              GPIO18 PWM0
