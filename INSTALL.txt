
(1) Edit and uncomment "blacklist i2c-bcm2708"
sudo vi /etc/modprobe.d/raspi-blacklist.conf

# blacklist spi and i2c by default (many users don't need them)

#blacklist spi-bcm2708
blacklist i2c-bcm2708

(2) Load kernel module
sudo modprobe spi-bcm2708

(3) Build dependencies
sudo apt-get -y install git-core
sudo apt-get update
sudo apt-get -y upgrade
git clone git://git.drogon.net/wiringPi
cd wiringPi
./build
cd ..
sudo apt-get -y install python-imaging python-imaging-tk python-pip python-dev git
sudo pip install spidev
sudo pip install wiringpi

(3) Shutdown pi
sudo shutdown -h now

(4) Connect jumper wires

 LCD Pin      Pi GPIO Pin #   Pi Pin Name
============= =============== ===============
 1 GND        6               GND
 2 VCC        1               3.3 V
 3 CLK        23              GPIO11 SPI0_SCLK
 4 DIN        19              GPIO10 SPI0_MOSI
 5 D/C        15              GPIO22
 6 CS         24              GPIO08 SPI0_CE0_N
 7 RST        11              GPIO17
 8 LED        12              GPIO18 PWM0

(Adafruit cobbler and jumper wires or another connection solution.)

(5) Power up Pi and run
sudo python magpiLCD.py
